# Expense Management System

A web-based Expense Management System built using the PERN stack (PostgreSQL, Express, React, Node.js).

## Core Features
- Expense submission with amount, currency, and category
- Role-based access (Employee, Manager, Admin)
- Approval workflows
- Audit logs

## Development Phases
- Phase 0: Project foundation and setup
- Phase 1: Expense CRUD
- Phase 2: Role-based approvals
- Phase 3: Rule-based approvals
- Phase 4: Audit logging

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: PostgreSQL

> This project is intentionally built without TypeScript, Prisma, Zod, or advanced abstractions in early phases to focus on core system design and SQL fundamentals.
